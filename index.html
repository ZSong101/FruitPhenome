Step 2: Scale and color calibration

? Print this sheet when you want a simple size reference in your images. Choose the marker size that fits your fruit, print it at the indicated physical size, cut it out if needed, and keep one marker flat and fully visible in each image. If a marker is detected, it can be used for automatic scale calibration.

Use an ArUco marker when you want automatic size calibration without entering a manual pixel-to-cm conversion. It is optional, but helpful when a known physical scale is needed.

ColorChecker reference: the ColorChecker is not necessary, but it is helpful for color calibration. Use the middle page of the Calibrite ColorChecker Passport Photo 2, the 24-patch page that looks like this reference image.

Scale priority: manual scale, then ArUco marker, then ColorChecker, then pixel units.

Augmented export options

These settings generate transformed training copies from approved masks. Preview uses the same augmentation pipeline at a smaller display resolution; export retains the configured training resolution.

Background augmentation mix

The four replacement probabilities form a background mix; any remaining fraction keeps the original background, and totals above 1 are normalized. Ordinary clutter and out-of-context objects are added independently and can coexist with any background. Set a probability to 0 to disable that family.
